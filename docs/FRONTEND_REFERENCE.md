# FRONTEND_REFERENCE

## 기술 레퍼런스 매핑

- MUI: https://mui.com/
  - 사용자/관리자 공통 AppBar + Sidebar + Card + Form + Dialog 컴포넌트의 기본 디자인 시스템으로 적용.
- MUI DataGrid: https://mui.com/x/react-data-grid/
  - 명세서 목록, 승인내역, 문의 큐, 문서 검토, 감사로그 등 운영형 테이블(페이지네이션/정렬/필터)의 핵심 패턴으로 적용.
- Ant Design(비교 참고): https://ant.design/docs/react/introduce/
  - 대시보드형 정보 밀도, 상태 배지 중심의 업무 UI 패턴 비교 참고.
- TanStack Table: https://tanstack.com/table/v8/docs/introduction
  - DataGrid 대체 가능 구조를 고려한 테이블 추상화 컴포넌트 설계 참고.
- Recharts: https://recharts.org/
  - 사용자 포털 대시보드의 월별 소비 추이 차트에 반영.
- CoreUI React Admin: https://coreui.io/product/free-react-admin-template/
  - 관리자 포털 레이아웃(좌측 네비 + 상단 바 + KPI 카드 + 큐 테이블) 구성에 반영.
- Refine 어드민 구성 참고: https://refine.dev/blog/react-admin-dashboard/
  - 목록→상세/처리→피드백 토스트 흐름, CRUD 중심 IA 구성에 반영.
- Material UI for Figma: https://mui.com/material-ui/design-resources/material-ui-for-figma/
  - 카드/폼/데이터테이블 기반 화면 컴포지션과 간격/타이포 체계 참고.
